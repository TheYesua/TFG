Declaración de accesibilidad

La aplicación se ha diseñado conforme a las WCAG 2.1 nivel AA.

Características incluidas

  • Enlace «Pasar al contenido principal» al inicio de cada página.
  • Foco visible reforzado (no depende solo del color).
  • Navegación completa por teclado: Tab/Mayús+Tab para moverse, Intro para activar, Esc para cerrar diálogos.
  • Etiquetas y encabezados semánticos (<header>, <nav>, <main>, <footer>, <h1>–<h4>).
  • Imágenes con texto alternativo y SVGs decorativos marcados con aria-hidden.
  • Formularios con label asociado, mensajes de error anunciados a lectores de pantalla y campos con aria-invalid cuando procede.
  • Contraste de texto mínimo 4.5:1 (3:1 en texto grande).
  • Información que no se transmite solo mediante color (iconos lingüísticos, texto y formas refuerzan los estados).
  • Diseño adaptable a móvil, tableta y escritorio; permite zoom hasta 200% sin pérdida de contenido.
  • Animaciones reducidas si el sistema indica prefers-reduced-motion: reduce.
  • Mapa web para navegación alternativa.

Limitaciones conocidas

Esta aplicación es un Trabajo de Fin de Grado en evaluación continua. Si detectas alguna barrera de accesibilidad, puedes comunicarla en la sección de Ayuda.
